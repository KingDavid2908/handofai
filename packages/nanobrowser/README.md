# HandofAI

HandofAI is an AI-powered browser extension for automating web tasks. Built on open-source foundations, it provides intelligent web navigation capabilities.

## Features

- **AI Web Navigation**: Automate web tasks with natural language
- **Data Extraction**: Extract data from websites automatically
- **Form Automation**: Fill forms intelligently
- **Privacy-Focused**: Runs locally in your browser
- **Multiple LLM Support**: Compatible with various LLM providers

## Installation

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Build: `pnpm build`
4. Load the extension in Chrome developer mode

## Usage

1. Configure your LLM API keys in the extension settings
2. Open the side panel
3. Enter your task in natural language
4. Let the AI navigate and complete your task
